# Window System

The portfolio features a complete window management system with drag, resize, minimize, maximize, and close functionality.

## WindowWrapper Component

All windows are wrapped with the `WindowWrapper` component located in `app/components/ui/WindowWrapper.tsx`.

## Window Features

### Drag Functionality

Windows can be dragged by their title bar:

```tsx
// From WindowWrapper.tsx:48-66
const dragControls = useDragControls();

const handleDragEnd = (event: any, info: any) => {
  if (!windowRef.current) return;
  
  const windowWidth = windowRef.current.offsetWidth;
  const windowHeight = windowRef.current.offsetHeight;
  
  // Constrain window to viewport bounds
  const newX = Math.min(
    Math.max(0, position.x + info.delta.x),
    window.innerWidth - windowWidth
  );
  const newY = Math.min(
    Math.max(0, position.y + info.delta.y),
    window.innerHeight - windowHeight
  );
  
  setPosition({ x: newX, y: newY });
};
```

**Usage in window header:**
```tsx
<motion.div
  className="h-12 bg-gradient-to-r from-gray-800/50 to-gray-900/50 cursor-move"
  onPointerDown={e => !isMaximized && dragControls.start(e)}
>
  {/* Window title and controls */}
</motion.div>
```

<Info>
  Dragging is disabled when the window is maximized or on mobile devices.
</Info>

**Resize constraints:**

- Minimum height: 300px
- Can resize from all 8 directions (4 edges + 4 corners)

## Responsive Behavior

### Mobile Adaptation

On mobile devices, windows automatically go full-screen:

```tsx
// From WindowWrapper.tsx:7-21
const useIsMobile = () => {
  const [isMobile, setIsMobile] = useState(false);
  
  useEffect(() => {
    const checkIsMobile = () => {
      setIsMobile(window.innerWidth < 768);
    };
    
    checkIsMobile();
    window.addEventListener('resize', checkIsMobile);
    return () => window.removeEventListener('resize', checkIsMobile);
  }, []);
  
  return isMobile;
};

// Usage
const shouldBeFullscreen = isMobile || isMaximized;
```

### Viewport Constraints

Windows stay within viewport bounds:

```tsx
// From WindowWrapper.tsx:68-81
useEffect(() => {
  const handleResize = () => {
    if (windowRef.current && !isMobile) {
      const rect = windowRef.current.getBoundingClientRect();
      setPosition(prev => ({
        x: Math.min(prev.x, window.innerWidth - rect.width),
        y: Math.min(prev.y, window.innerHeight - rect.height),
      }));
    }
  };
  
  window.addEventListener('resize', handleResize);
  return () => window.removeEventListener('resize', handleResize);
}, [isMobile]);
```

## Animation System

Windows use Framer Motion for smooth animations:

```tsx
// From WindowWrapper.tsx:84-119
<motion.div
  initial={{ opacity: 0 }}
  animate={{ opacity: 1 }}
  exit={{ opacity: 0 }}
  className="fixed inset-0 z-50"
>
  {/* Backdrop */}
  <div className="absolute inset-0 bg-black/20 backdrop-blur-sm" onClick={onClose} />
  
  {/* Window */}
  <motion.div
    ref={windowRef}
    initial={{ opacity: 0, scale: isMobile ? 1 : 0.98 }}
    animate={{
      opacity: 1,
      scale: 1,
      x: shouldBeFullscreen ? 0 : position.x,
      y: shouldBeFullscreen ? 0 : position.y,
      width: shouldBeFullscreen ? '100%' : size.width,
      height: shouldBeFullscreen ? '100%' : size.height,
    }}
    transition={{
      type: 'spring',
      stiffness: 300,
      damping: 25,
      mass: 0.5,
    }}
    drag={!shouldBeFullscreen}
    dragControls={dragControls}
    dragConstraints={constraintsRef}
    dragMomentum={false}
    dragElastic={0}
    onDragEnd={handleDragEnd}
  >
    {children}
  </motion.div>
</motion.div>
```

## Best Practices

<Steps>
  <Step title="Use WindowWrapper">
    Always wrap your window content with `WindowWrapper` for consistent behavior
  </Step>
  
  <Step title="Manage State Locally">
    Each window manages its own maximize/minimize state
  </Step>
  
  <Step title="Provide Initial Sizes">
    Set appropriate `initialWidth` and `initialHeight` based on content
  </Step>
  
  <Step title="Handle Mobile">
    Ensure content is scrollable and works well in full-screen mode
  </Step>
  
  <Step title="Use Custom Scrollbar">
    Apply `custom-scrollbar` and `mobile-hide-scrollbar` classes for consistent UX
  </Step>
</Steps>
