# Terminal Mode

Terminal Mode provides a fully functional command-line interface for navigating the portfolio using Unix-like commands.

## Overview

Toggle to Terminal Mode using the floating button at the bottom-right of the screen. The terminal provides:

- Full filesystem navigation
- Command history with arrow keys
- Tab auto-completion
- Built-in portfolio commands
- Math expression evaluation

## Getting Started

### First Visit

When you first enter Terminal Mode, you'll be prompted to enter your name:

```bash

Welcome John Doe! Type 'help' to see available commands.
```

## Advanced Features

### Command History Navigation

Navigate through previous commands using arrow keys:

```tsx
// From Terminal.tsx:107-131
const handleKeyPress = (e: KeyboardEvent<HTMLInputElement>) => {
  if (e.key === 'Enter') {
    processCommand(e);
  } else if (e.key === 'ArrowUp') {
    e.preventDefault();
    if (historyIndex < commandHistory.length - 1) {
      const newIndex = historyIndex + 1;
      setHistoryIndex(newIndex);
      setInput(commandHistory[commandHistory.length - 1 - newIndex]);
    }
  } else if (e.key === 'ArrowDown') {
    e.preventDefault();
    if (historyIndex > 0) {
      const newIndex = historyIndex - 1;
      setHistoryIndex(newIndex);
      setInput(commandHistory[commandHistory.length - 1 - newIndex]);
    } else if (historyIndex === 0) {
      setHistoryIndex(-1);
      setInput('');
    }
  } else if (e.key === 'Tab') {
    e.preventDefault();
    autoComplete();
  }
};
```

<Tip>
  Press ↑ to cycle backward through history, ↓ to cycle forward. Press Enter on any command to execute it again.
</Tip>

### Tab Auto-Completion

Press `Tab` to auto-complete file and directory names:

```tsx
// From Terminal.tsx:133-146
const autoComplete = () => {
  const parts = input.trim().split(' ');
  const lastPart = parts[parts.length - 1];
  const currentPath = getCurrentPath();
  
  if (currentPath.contents) {
    // Find matches that start with the input
    const matches = Object.keys(currentPath.contents)
      .filter(name => name.startsWith(lastPart));
    
    // Auto-complete if only one match
    if (matches.length === 1) {
      parts[parts.length - 1] = matches[0];
      setInput(parts.join(' '));
    }
  }
};
```

**Example:**
```bash
guest@portfolio:~$ cat ab<Tab>
guest@portfolio:~$ cat about.txt
```

## Command Processing

Commands are processed in the `processCommand` function:

```tsx
// From Terminal.tsx:148-455
const processCommand = async (e: KeyboardEvent<HTMLInputElement>) => {
  e.preventDefault();
  
  const fullCommand = input.trim();
  const parts = fullCommand.split(' ');
  const command = parts[0].toLowerCase();
  const args = parts.slice(1);
  
  // Add to command history
  if (fullCommand) {
    setCommandHistory(prev => [...prev, fullCommand]);
    setHistoryIndex(-1);
  }
  
  let output;
  
  switch (command) {
    case 'ls':
      // List directory contents
      break;
    case 'cd':
      // Change directory
      break;
    case 'cat':
      // Display file contents
      break;
    // ... more commands
  }
  
  // Add command and output to history
  setCommands(prevCommands => [...prevCommands, {
    id: Date.now(),
    input: fullCommand,
    time: getFormattedTime(),
    output,
    directory: currentDirectory,
  }]);
  
  setInput('');
};
```

## Terminal Clock

A live clock displays at the top of the terminal:

```tsx
// From Terminal.tsx:6-13
const TerminalClock = () => {
  const [time, setTime] = useState(getFormattedTime());
  
  useEffect(() => {
    const timer = setInterval(() => setTime(getFormattedTime()), 1000);
    return () => clearInterval(timer);
  }, []);
  
  return <div className="text-xs mb-2 sm:mb-4 opacity-70">Last login: {time}</div>;
};
```
