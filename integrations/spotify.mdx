## Overview

The Spotify integration provides real-time music data from your Spotify account, including:

- Currently playing track with live updates
- Top tracks and artists
- User playlists
- Recently played tracks
- Top genres visualization

All data is fetched through the Spotify Web API using OAuth 2.0 authentication with refresh tokens.

### Top Tracks

**Endpoint:** `GET /api/spotify/top-tracks`

**Location:** `app/api/spotify/top-tracks/route.ts:4`

Returns your top 10 tracks from the short-term time range.

<CodeGroup>
```typescript Request
const response = await fetch('/api/spotify/top-tracks');
const data = await response.json();
```

```json Response
{
  "tracks": [
    {
      "title": "Track Name",
      "artist": "Artist Name",
      "url": "https://open.spotify.com/track/...",
      "albumImageUrl": "https://i.scdn.co/image/..."
    }
  ]
}
```
</CodeGroup>

**Implementation:** `lib/spotify.ts:51`

```typescript
export async function getTopTracks() {
  const { access_token } = await getAccessToken();
  
  const response = await fetch(
    `${TOP_TRACKS_ENDPOINT}?limit=10&time_range=short_term`,
    {
      headers: { Authorization: `Bearer ${access_token}` },
    }
  );
  
  const { items } = await response.json();
  return {
    tracks: items.map((track: any) => ({
      title: track.name,
      artist: track.artists.map((_artist: any) => _artist.name).join(', '),
      url: track.external_urls.spotify,
      albumImageUrl: track.album.images[0].url,
    })),
  };
}
```

## Authentication Flow

The Spotify integration uses OAuth 2.0 with refresh tokens for authentication.

### Authorization URL

**Location:** `app/api/spotify/auth.ts:5`

```typescript
const SPOTIFY_AUTH_URL = `https://accounts.spotify.com/authorize
  ?client_id=${client_id}
  &response_type=code
  &redirect_uri=${redirect_uri}
  &scope=user-read-recently-played%20user-top-read%20user-read-currently-playing%20user-read-playback-state`;
```

### Callback Route

**Endpoint:** `GET /api/spotify/callback`

**Location:** `app/api/spotify/callback/route.ts:3`

Handles the OAuth callback and exchanges the authorization code for tokens.

```typescript
export async function GET(request: Request) {
  const { searchParams } = new URL(request.url);
  const code = searchParams.get('code');

  if (!code) {
    return NextResponse.json({ error: 'No code provided' }, { status: 400 });
  }

  const tokenResponse = await fetch('https://accounts.spotify.com/api/token', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/x-www-form-urlencoded',
      Authorization: `Basic ${Buffer.from(
        `${process.env.SPOTIFY_CLIENT_ID}:${process.env.SPOTIFY_CLIENT_SECRET}`
      ).toString('base64')}`,
    },
    body: new URLSearchParams({
      grant_type: 'authorization_code',
      code,
      redirect_uri: 'http://localhost:3000/api/spotify/callback',
    }),
  });

  const data = await tokenResponse.json();
  
  // Returns the refresh_token to be saved
}
```

### Token Refresh

**Location:** `lib/spotify.ts:11`

```typescript
async function getAccessToken() {
  const response = await fetch(TOKEN_ENDPOINT, {
    method: 'POST',
    headers: {
      Authorization: `Basic ${basic}`,
      'Content-Type': 'application/x-www-form-urlencoded',
    },
    body: new URLSearchParams({
      grant_type: 'refresh_token',
      refresh_token: refresh_token!,
    }),
  });

  return response.json();
}
```

## Setup and Configuration

### Environment Variables

Add these to your `.env.local` file:

```bash
SPOTIFY_CLIENT_ID=your_client_id
SPOTIFY_CLIENT_SECRET=your_client_secret
SPOTIFY_REFRESH_TOKEN=your_refresh_token
```

### Getting Your Credentials

1. **Create a Spotify App:**
   - Go to [Spotify Developer Dashboard](https://developer.spotify.com/dashboard)
   - Create a new app
   - Note your Client ID and Client Secret

2. **Set Redirect URI:**
   - In your app settings, add: `http://localhost:3000/api/spotify/callback`

3. **Get Refresh Token:**
   - Visit the authorization URL (defined in `app/api/spotify/auth.ts:5`)
   - Authorize the app
   - You'll be redirected to `/api/spotify/callback` with your refresh token
   - Copy the token and add it to your `.env.local`

### Required Scopes

**Location:** `app/api/spotify/auth.ts:5`

- `user-read-recently-played` - Recently played tracks
- `user-top-read` - Top tracks and artists
- `user-read-currently-playing` - Now playing
- `user-read-playback-state` - Playback state

## Error Handling

All API routes include proper error handling:

```typescript
export async function GET() {
  try {
    const response = await getNowPlaying();
    return NextResponse.json(response);
  } catch (error) {
    return NextResponse.json(
      { error: 'Error fetching now playing' },
      { status: 500 }
    );
  }
}
```

Components handle errors gracefully and display fallback states when data cannot be fetched.
